Minecraft Structure Export
==================================================

Structure size: 8x8x8
Total blocks: 60
Offset: (0, 100, 0)

HOW TO USE:
--------------------------------------------------
Method 1: Copy-paste commands
  1. Open structure.mcfunction
  2. Copy all commands
  3. In Minecraft, press T to open chat
  4. Paste and run commands (may need to do in batches)

Method 2: Use as datapack function
  1. Create a datapack folder structure
  2. Place structure.mcfunction in data/namespace/functions/
  3. Load datapack and run with /function namespace:structure

Note: Large structures may cause lag. Consider splitting into chunks.
